# discount-alerter

Toy project to try https://github.com/cgrand/enlive library.

Checks game prices in steam and report discount status.

## Installation

lein uberjar

## Usage

    $ java -jar discount-alerter-0.1.0-standalone.jar [args]


## Examples
For games Dark Souls (id=211420) and Crusader Kings II (id=203770)

$ java -jar discount-alerter-0.1.0-standalone.jar 211420 203770
